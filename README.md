Este proyecto presenta el desarrollo de un modelo de aprendizaje supervisado 
basado en árboles de decisión para la predicción del tiempo de viaje en la 
red de Transmilenio de Bogotá.

En el contexto del transporte público masivo, es fundamental predecir con 
precisión el tiempo de desplazamiento entre estaciones para:
- Optimizar la experiencia del usuario
- Mejorar la eficiencia operativa del sistema
- Facilitar la planificación de rutas

Este proyecto implementa los conceptos teóricos del Capítulo 17 del libro 
"Inteligencia artificial: métodos, técnicas y aplicaciones" (Palma Méndez, 2008),
aplicándolos específicamente a árboles de decisión para regresión.
